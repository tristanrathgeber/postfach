# Postfach v0.1 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: superpowers:executing-plans (Backend-Track inline, TDD) + ein paralleler Frontend-Agent gegen `docs/api-contract.md`. Steps use checkbox (`- [ ]`) syntax.

**Goal:** Lauffähige lokale Notion-Mail-artige App: Multi-Account-IMAP, AI-Views, sicheres Lesen, AI-Entwürfe, Senden, Triage, Cmd+K — inkl. Demo-Modus.

**Architecture:** FastAPI (127.0.0.1:8722) mit eigener Mail-Schicht (imapclient+smtplib+nh3) und email-agent als Path-Dependency für Classifier/Drafter/Style/LLM; React/Vite/Tailwind-Frontend als Static-Build. Vertrag: `docs/api-contract.md` (eingefroren).

**Tech Stack:** Python 3.12/uv/FastAPI/nh3/imapclient/pytest+httpx · Vite/React/TS/Tailwind/TanStack Query/cmdk.
